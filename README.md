# McCann Dillon Jaffe and Lamb

Homepage demo and editorial site for McCann Dillon Jaffe and Lamb, LLC, a Pennsylvania trial firm with offices in Philadelphia, Wilmington, Haddon Township, and Jenkintown.

## Demo

- `index.html`, homepage with hero, verdict ledger, 6 editorial cards, 6 practice areas, 7 attorney bench, E-E-A-T pillars, 5-step approach, CTA, footer
- `article-i95-verdicts.html`, full data-journalism piece by Robert E. McCann analyzing 73 I-95 corridor verdicts 2018 to 2025

## Brand

- Tagline: TWENTY-FIVE YEARS. ONE TRIAL RECORD.
- Colors: Independence Navy, Liberty Brass, Alert Crimson, Philly Newsprint, Keystone Slate
- Type: Anton (display), Inter (body), JetBrains Mono (eyebrow), Source Serif 4 (article body)
- Logo: Bracket wordmark, dark background only (light-bg version pending)
- Core truth: McCann prepares every case like it's going to trial.
